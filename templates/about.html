About Music Analysis System

Project Overview

A music review analysis system built with Flask, processing over 80,000 reviews from Rate Your Music. The reviews don't include an album that was reviewed, so the system is assigning predicted albums based on context. The dataset includes second SQL table containg data on 5000 most rated albums on RYM.
